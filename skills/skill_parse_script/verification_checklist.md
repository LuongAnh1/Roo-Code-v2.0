# CHECKLIST KIỂM CHỨNG: MAP DỮ LIỆU KỊCH BẢN

Trước khi báo cáo hoàn thành Pha 1, Agent BẮT BUỘC phải đối chiếu dữ liệu vừa trích xuất với danh sách kiểm tra này. Nếu có bất kỳ mục nào chưa đạt (NO), phải thực hiện lại việc đọc file.

| ID | Tiêu chí kiểm tra | Kết quả (YES/NO) | Ghi chú |
|:---|:---|:---:|:---|
| 1 | **Duration Check (Hook)?** Dòng thoại Scene 1 (Hook) có **dưới 30 từ** không? | | **Dừng lại báo cáo nếu NO** |
| 2 | **Bỏ qua Header?** Đã xác nhận không lấy nội dung dòng 1 (Hook, Body...) làm kịch bản chưa? | | |
| 3 | **Ghép cặp 2 hàng?** Đã đảm bảo Thoại nằm ở dòng lẻ (2, 4...) và Hình ảnh nằm ngay dòng chẵn phía dưới (3, 5...)? | | |
| 4 | **Đủ 4 phân cảnh?** Có đủ 4 cột tương ứng: Hook, Main Body, Takeaways, CTA chưa? | | |
| 5 | **Dữ liệu rỗng?** Có ô nào bị thiếu thông tin Thoại hoặc Hình ảnh không? | | |
| 6 | **Ước tính Hook?** Dòng thoại Scene 1 có đảm bảo đọc kịp trong ~5 giây không (dưới 30 từ)? | | |
| 7 | **Kiểm định Manim?** Mô tả hình ảnh có chứa yếu tố "không tưởng" đối với Manim (ví dụ: ảnh thật, người thật) không? | | |
